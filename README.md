# Dynamical Mass of a Galaxy Cluster using SDSS Data

This was an assignment given by Dr.Anand Narayanan IIST, for the course Galaxies in 2023.

In this project, we aim to compute the dynamical mass of a galaxy cluster using spectroscopic data from the Sloan Digital Sky Survey (SDSS). The mass is estimated using velocity dispersion and the virial theorem.

## Overview

Galaxy clusters are the largest gravitationally bound structures in the universe. Their total mass can be estimated dynamically using the velocity distribution of member galaxies.

The following steps are performed,

Selection of cluster member galaxies using redshift cuts

Conversion of redshift to line-of-sight velocity

Calculation of velocity dispersion

Estimation of cluster radius

Computation of dynamical mass using the virial theorem
